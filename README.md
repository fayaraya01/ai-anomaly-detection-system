# 🚀 AI Agent-Based Anomaly Detection System

## 📌 Overview
This project detects anomalies in data using machine learning and explains them using an AI agent. It provides real-time monitoring through a dashboard.

## ⚙️ Features
- Anomaly detection using Isolation Forest
- AI-based explanation system
- FastAPI backend
- Streamlit dashboard
- Alert system

## 🛠️ Tech Stack
- Python
- FastAPI
- Streamlit
- Scikit-learn

## ▶️ How to Run

### Backend
uvicorn backend.main:app --reload

### Frontend
streamlit run frontend/app.py

## 🎯 Use Cases
- Fraud detection
- Cybersecurity monitoring
- IoT anomaly detection
